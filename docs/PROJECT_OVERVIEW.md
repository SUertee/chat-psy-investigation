# 危机评估培训实验系统 - 项目总览

## 📋 项目简介

这是一个完整的心理学实验系统，用于研究AI辅助危机评估培训的效果。系统采用极简设计，专注于功能实现，让参与者无需下载任何软件即可通过网页链接参与实验。

## 🎯 核心功能

### ✅ 已实现功能

1. **完整的实验流程**
   - 知情同意 → 随机分组 → 个人信息 → 前测问卷 → 视频学习 → 练习/观摩 → 后测问卷 → 二次练习 → 危机评估 → 问卷3 → AI反馈 → 问卷4

2. **随机分组系统**
   - 1:1随机分配实验组/对照组
   - 自动记录分组结果

3. **问卷系统**
   - 前测问卷（个人背景、WIS量表42题、知识测试10题）
   - 后测问卷（参与度、WIS量表、知识测试、状态焦虑）
   - 第三次问卷（可接受性、开放性问题）
   - 第四次问卷（状态检查）

4. **AI交互系统**
   - 实验组：与AI虚拟来访者对话（微信聊天样式）
   - 对照组：PDF观摩学习
   - 10分钟倒计时，可主动结束

5. **AI督导反馈**
   - 自动分析对话记录
   - 从4个维度提供评估反馈
   - 使用API生成个性化反馈

6. **数据收集**
   - 记录每个步骤的时间戳
   - 自动保存聊天记录
   - 生成Excel文件（本地+服务器）
   - 包含所有实验数据

7. **进度显示**
   - 顶部进度条显示完成百分比
   - 清晰的时间倒计时

## 🚀 使用方法

### 快速部署（3步）

1. **上传文件**
   ```bash
   # 将所有文件放在Web服务器目录
   cp * /var/www/html/experiment/
   ```

2. **配置API**
   ```javascript
   // 编辑 src/config/config.js
   VOLCENGINE_API_KEY: "您的实际API密钥"
   ```

3. **访问实验**
   ```
   http://your-server-ip/experiment/public/index.html
   ```

### 本地测试
```bash
python -m http.server 8000
# 访问 http://localhost:8000/public/test.html
```

## 🔧 定制化修改

### 修改问卷内容
在 `config.js` 的 `QUESTIONNAIRES` 对象中修改：

```javascript
const QUESTIONNAIRES = {
    pretest: {
        title: "前测问卷",
        questions: [
            // 添加/修改问题
            { id: "age", type: "number", text: "年龄", required: true },
            // ...
        ]
    }
};
```

### 修改AI角色设定
在 `config.js` 的 `AI_PROMPTS` 中修改：

```javascript
const AI_PROMPTS = {
    EXPERIMENTAL_CLIENT: `您的AI角色设定...`,
    SECOND_CLIENT: `第二次对话的角色设定...`,
    SUPERVISOR: `AI督导的评估标准...`
};
```

### 修改实验设置
在 `config.js` 的 `EXPERIMENT_CONFIG` 中修改：

```javascript
const EXPERIMENT_CONFIG = {
    COUNTDOWN_TIME: 15 * 60 * 1000,  // 改成15分钟
    DATA_UPLOAD_URL: "您的服务器地址",
    // ...
};
```
