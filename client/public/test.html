📊 功能说明

  • 实验流程：知情同意 → 随机分组 → 问卷 → 视频学习 → 练习/观摩 → 评估 → AI反馈
  • 数据收集：自动保存Excel文件，包含时间戳、问卷数据、聊天记录等
  • AI/配对交互：实验组与AI对话，对照组可接入配对聊天室，最后获得AI督导反馈
  • 进度显示：顶部进度条显示当前完成百分比

🚀 开始实验

系统检查完成后，点击下方按钮开始实验：

开始实验 进入对照组 Debug 聊天室

该按钮会跳过正式流程，直接进入对照组配对聊天室。请至少打开两个窗口用于配对。
